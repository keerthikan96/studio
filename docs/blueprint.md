# **App Name**: StaffSync

## Core Features:

- Admin Dashboard: A central hub for administrators to manage staff profiles and system settings.
- Staff Profile Creation via Resume: Admins can upload a resume, and an AI tool automatically extracts relevant information to pre-fill staff profile forms, reducing manual data entry.
- Staff Invitation: Admins can invite new staff members via email with a unique, secure link.
- Secure Password Setup: New staff members are guided through a secure password creation process upon first login.
- Profile Management: Staff members can view and update their personal information and profile details.
- Role-Based Access: Different levels of access are granted based on user role (admin or staff).

## Style Guidelines:

- Primary color: Deep purple (#673AB7), for professionalism and innovation.
- Background color: Light gray (#F0F2F5), providing a clean and modern backdrop.
- Accent color: Teal (#009688), to highlight interactive elements and important actions.
- Body and headline font: 'PT Sans', a humanist sans-serif for a blend of modern and approachable feel.
- Use clean, line-based icons to represent staff roles, departments, and functions.
- Prioritize a clean and intuitive layout with clear sections for different management tasks.
- Subtle transitions and animations to provide feedback on user interactions.
